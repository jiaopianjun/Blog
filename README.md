###  基于Vue的个人博客，前端正在开发当中...

#### 功能：
    1.兼容PC 和 移动端
    2.可以进行项目分类和子标签分类
    3.后台管理和文章编写发送

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
